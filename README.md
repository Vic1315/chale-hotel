Projeto Final do Curso de Des. HTML - Prof Rafael Jesus

Site Chalé Hotel

Usadas como tecnologias HTML5, CSS e JavaScript.

O site utiliza recursos como formulários, acessibilidade, imagens e layout líquido.

O site apresenta o hotel e suas características, permite contato através de formulários e reservas a partir do site selecionando dados de check-in e check-out.

Para executar, clique em <>Código e baixe o projeto como .zip. Extraia os arquivos e abra o index.html no seu navegador.
